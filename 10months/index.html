💡 V8 做法提示

  • 🌠 锡纸香肠：肉泥+蔬菜+淀粉搅匀 → 裱花袋挤在锡纸上 → 卷长条两头拧紧 → 蒸15min → 冷却剥锡纸切小段
  • 🥚 蛋卷：蔬菜焯水切极碎末 → 鸡蛋+面粉+菜末搅匀 → 平底锅小火摊薄蛋皮 → 趁热卷起切2cm段
  • 🍲 蛋羹：鸡蛋打散加1.5倍温水过滤 → 混入虾皮/紫菜末 → 蒸12min（盖保鲜膜防气孔）
  • 🧀 肉丸：肉泥+蒸软蔬菜泥/末+淀粉搅匀 → 搓1.5cm小丸 → 蒸10-12min
  • 🍱 软饼：蔬菜焯熟切极碎末 → 鸡蛋+面粉+菜末搅匀 → 平底锅小火摊小饼
  • 🍟 馄饨：煮熟捞出剪小块 → 一周2次(周三+周六) → 15天保质期内用完
  • 🌸 肝粉用法：出锅前拌入烩饭/面条 → 搭配番茄/南瓜甜味掩盖腥味
  • 🥕 蔬菜条：西葫芦擦丝挤水+肉泥+面粉+蛋黄 → 挤长条(≈5cm) → 小火煎金黄
  • 💡 油轮换：核桃油/亚麻籽油/紫苏油/橄榄油/牛油果油/芝麻油，每天不同，脂肪酸均衡
  • 🍡 馒头：买的（原味+南瓜小米红枣），每日2个(≈30g)，纳入营养计算
